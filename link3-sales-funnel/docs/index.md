# Link3 Sales Funnel AI Closer

AI-powered sales lead qualification and follow-up assistant for Link3 sales workflows.

## Run

```bash
streamlit run link3-sales-funnel/Link3_Sales_Funnel_AI_Closer.py
```

## What It Does

- Scores B2B and B2C leads.
- Detects funnel stage, deal temperature, urgency, and likely deal size.
- Recommends Link3 package fit, such as dedicated internet, SME shared fiber, static public IP, SLA support, backup link, or home fiber packages.
- Generates a ready-to-send reply, objection response, follow-up plan, and CRM-ready note.
- Shows a transparent rule-based score beside the AI result for comparison.
